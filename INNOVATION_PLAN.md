# 설문 시스템 개선 및 데이터 표준화 제안서

본 문서는 사회연대기금 설문조사 시스템의 통계 분석 신뢰도와 운영 효율성을 높이기 위해 진행된 데이터 표준화 작업의 내용과 근거를 정리한 보고서입니다.

## 1. 개선 배경 및 필요성

기존 설문 설계의 주관식(자유 텍스트) 응답 방식은 다음과 같은 문제점이 있었습니다.

*   **통계 분석의 어려움**: 동일한 의미의 응답이라도 오타, 띄어쓰기, 표현 방식의 차이(예: '대표', '대표이사', '사장')로 인해 데이터가 분산되어 정확한 정량적 통계 산출이 불가능했습니다.
*   **수동 데이터 정제 비용 발생**: 수집된 데이터를 분석 가능한 형태로 가공하기 위해 수만 개의 응답을 사람이 직접 분류해야 하는 막대한 리소스 낭비가 예상되었습니다.
*   **분석의 일관성 결여**: 응답자의 주관에 따라 답변의 범위와 깊이가 달라져, 설문의 일관된 목적을 달성하는 데 한계가 있었습니다.

## 2. 주요 개선 내용 (표준화 방안)

주요 통계 지표로 활용되는 5가지 핵심 항목을 텍스트 입력에서 **표준 객관식(라디오 버튼)**으로 전환하였습니다.

| 항목 | 기존 방식 | 개선 방식 (표준 카테고리) |
| :--- | :--- | :--- |
| **직위** | 직접 입력 | 대표 / 이사·임원 / 팀장·매니저 / 실무자 / 기타 |
| **근무지** | 직접 입력 | 광명시 관내 6개 주요 동 / 광명시 외 |
| **연간 매출액** | 직접 입력 | 5천만 미만 ~ 10억 이상 (6개 구간 표준화) |
| **사업 범위** | 직접 입력 | 관내 동 단위 / 시 전체 / 경기도 / 수도권 / 전국 / 해외 |
| **필요 자금 규모** | 시간 단위 (오류) | 1,000만원 미만 ~ 1억원 이상 (금액 단위 정상화) |

*※ '기타' 옵션을 제공하여 표준 카테고리에 포함되지 않는 특수 사례에 대한 유연성도 확보하였습니다.*

## 3. 기대 효과 (혁신 가치)

이번 표준화 작업을 통해 다음과 같은 혁신적 가치를 확보할 수 있습니다.

1.  **데이터 무결성 확보**: 수집 단계에서부터 데이터 형식이 통일되어 오차 없는 통계 결과 도출이 가능합니다.
2.  **실시간 통계 가시화**: Google Sheet와 연동된 대시보드를 통해 데이터 정제 과정 없이 즉시 설문 현황을 분석할 수 있습니다.
3.  **정책 근거의 신뢰도 향상**: 표준화된 구간별 데이터를 바탕으로 광명시 사회연대기금 설치 및 운용 방안 수립을 위한 객관적이고 강력한 통계적 근거를 제시할 수 있습니다.
4.  **사용자 경험 개선**: 응답자가 고민하여 텍스트를 입력하는 대신, 명확한 선택지를 통해 설문에 참여함으로써 응답 피로도를 줄이고 제출 완료율을 높였습니다.
